Daily Expense Tracker (CLI)

This is a simple Command Line Interface (CLI) application written in Python to help track daily expenses. It uses an SQLite database to securely store the expense records.

Features
Add New Expense:** Easily record daily expenses by entering the Date, Amount, and Description.
View Total Expenses:** View the complete expense history and automatically calculate the total amount spent.
Secure Storage:** All records are securely saved in a local database (`expenses.db`) rather than a simple text file.

Technologies Used
Python 3
SQLite3 (Built-in Python Database)
